    Tema 8: Fundamentos del Lenguaje DAX

    Capítulo 1 – Las Categorías de las Funciones DAX

    21. Las Categorías de las Funciones DAX

    ¿Qué es DAX?

    DAX es un lenguaje de fórmulas usado en Power BI que significa Data Analysis Expressions o Expresiones de Análisis de Datos. También es utilizado en otros productos de Microsoft, como Power Pivot y SSAS Tabular.

    Las funciones que proporciona este lenguaje tienen funcionalidades idénticas y los mismos nombres que las funciones de Excel. Así, se consigue que los usuarios ya estén familiarizados con la nomenclatura.

    31. Las Categorías de las Funciones DAX

    Sin embargo, estas funciones se modificaron para usar los tipos de datos de DAX y así trabajar con tablas y columnas. También proporcionan una amplia variedad de funciones especializadas (para propósitos específicos ofrecidos por DAX):

    • Búsquedas basadas en relaciones
    • La capacidad de iterar sobre una tabla para realizar cálculos recursivos
    • Cálculos que usan la inteligencia de tiempo
    • Cambiar el contexto de los objetos visuales

    41. Las Categorías de las Funciones DAX

    Cuenta con conceptos sencillos y eficaces. Se basa en patrones y conceptos de programación únicos, y por esa razón puede costar su comprensión y ponerlo en práctica en su totalidad.

    Es posible que los métodos tradicionales de aprendizaje de lenguajes informáticos no sean la forma más adecuada de iniciarse en este lenguaje.

    51. Las Categorías de las Funciones DAX

    Existen diferentes tipos de funciones en el lenguaje DAX, que se agrupan de forma que cada uno de estos grupos, tienen unas finalidades específicas en el desarrollo de medidas y columnas calculadas.

    Los conceptos de medidas y columnas calculadas, se detallarán dentro de este tema. A continuación, se detalla y se explica cada uno de los grupos de funciones de los que dispone DAX.

    61. Las Categorías de las Funciones DAX

    Funciones de fecha y hora:

    Las funciones de fecha y hora de DAX son similares a las funciones de fecha y hora de Excel.

    71. Las Categorías de las Funciones DAX

    Funciones de filtro:

    Devuelven una columna o una tabla o valores vinculados a la fila actual. Puede utilizar las funciones de filtro de DAX para devolver tipos específicos de datos, buscar valores en tablas relacionadas y filtrar por valores relacionados. También permiten manipular el contexto de los datos para crear cálculos dinámicos.

    81. Las Categorías de las Funciones DAX

    Funciones financieras:

    Las funciones financieras se usan en fórmulas que realizan cálculos financieros, como el valor neto presente y la tasa de devolución. Estas funciones son similares a las funciones financieras usadas en Microsoft Excel.

    91. Las Categorías de las Funciones DAX

    Funciones de información:

    Estas funciones, sirven para examinar la celda o fila proporcionada como argumento y devuelven como resultado si el valor coincide con el tipo esperado.

    101. Las Categorías de las Funciones DAX

    Funciones lógicas:

    Devuelven información sobre los valores de una expresión. Por ejemplo, la función DAX TRUE le permite saber si una expresión que evalúa devuelve un valor TRUE.

    111. Las Categorías de las Funciones DAX

    Funciones matemáticas y trigonométricas:

    Las funciones matemáticas y trigonométricas de DAX son muy similares a las funciones matemáticas y trigonométricas de Excel.

    121. Las Categorías de las Funciones DAX

    Funciones de relación:

    Estas funciones proporcionan información sobre las relaciones entre las tablas.

    131. Las Categorías de las Funciones DAX

    Funciones estadísticas:

    Devuelven información sobre estadísticas descriptivas, como el promedio, la varianza y la desviación estándar.

    141. Las Categorías de las Funciones DAX

    Funciones de manipulación de tablas:

    Estas funciones permiten manipular tablas para que se adapten a sus necesidades.

    151. Las Categorías de las Funciones DAX

    Funciones de texto:

    Las funciones de texto en DAX se usan para devolver información sobre texto en una columna.

    161. Las Categorías de las Funciones DAX

    Funciones de ingeniería:

    Estas funciones permiten realizar cálculos en bases de datos multidimensionales.

    171. Las Categorías de las Funciones DAX

    Recursos Adicionales

    La documentación detallada de DAX está disponible en Microsoft Docs.

    Capítulo 2 – Operadores DAX

    21. Introducción a Operadores DAX

    Los operadores DAX se utilizan para realizar comparaciones, cálculos aritméticos y trabajar con cadenas de texto.

    31. Tipos de Operadores

    Existen varios tipos de operadores en DAX, incluyendo:

    • Operadores aritméticos para realizar operaciones matemáticas básicas.
    • Operadores de comparación para comparar valores y obtener resultados lógicos (TRUE/FALSE).
    • Operadores de concatenación de texto para unir cadenas.
    • Operadores lógicos para combinar expresiones y generar resultados lógicos.

    41. Importancia de Operadores en Programación

    Los operadores son esenciales en programación y se utilizan para realizar diversas operaciones. Es importante comprender cómo funcionan y cuándo aplicarlos.

    Capítulo 3 – Diferencias entre Columnas Calculadas y Medidas

    21. Conceptos Básicos

    En Power BI, las columnas calculadas y las medidas son elementos clave para realizar cálculos personalizados en los informes.

    31. Creación de Columnas Calculadas

    Puede crear columnas calculadas desde el panel de 'Informe' en Power BI, en la pestaña 'Modelado'.

    41. Ejemplos de Columnas Calculadas

    Las columnas calculadas permiten realizar cálculos a nivel de fila, utilizando fórmulas DAX para cada fila de la tabla.

    51. Ventajas y Limitaciones de Columnas Calculadas

    Las columnas calculadas tienen ventajas, como la evaluación a nivel de filas, pero también tienen limitaciones en términos de agregación y visualización.

    61. Creación de Medidas

    Las medidas son creadas desde el panel de 'Informe' en Power BI y se utilizan para realizar cálculos a nivel de totales o subtotales.

    71. Diferencias Clave

    La principal diferencia entre columnas calculadas y medidas es cómo se evalúan. Las columnas calculadas se evalúan a nivel de filas, mientras que las medidas se adaptan al contexto visual.

    Capítulo 4 – Medidas Rápidas

    21. Introducción a Medidas Rápidas

    Las medidas rápidas son un recurso facilitador que permite a los usuarios crear medidas sin tener conocimientos profundos de DAX.

    31. Tipos de Cálculos en Medidas Rápidas

    Las medidas rápidas permiten realizar una variedad de cálculos, incluyendo agregados, filtros, inteligencia de tiempo, totales, operaciones matemáticas y funciones de texto.

    41. Creación y Configuración

    La creación de medidas rápidas es un proceso simple que implica seleccionar el tipo de cálculo deseado y los campos necesarios.

    51. Adaptación de Medidas Rápidas

    Es posible ajustar las medidas rápidas generadas según las necesidades específicas del usuario o del informe.

    61. Consideraciones y Visualización

    Las medidas rápidas se pueden utilizar en objetos visuales y su fórmula DAX asociada se muestra en la visualización.

    Capítulo 5 – Conclusiones y Recursos Adicionales

    21. Equilibrio en el Uso de Recursos

    Es importante encontrar un equilibrio en el uso de columnas calculadas, medidas y medidas rápidas para optimizar el rendimiento y la eficacia.

    31. Recursos para Profundizar

    Para ampliar los conocimientos sobre DAX, se pueden consultar los recursos adicionales disponibles en la documentación oficial de DAX en Microsoft Docs.
