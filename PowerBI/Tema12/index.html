    5. Funciones comunes y recomendadas COUNTROWS: Contar el número de filas de una tabla. Ejemplo: Cuenta Filas = COUNTROWS('Ventas Videojuegos') Funciones comunes y recomendadas

    6. Funciones comunes y recomendadas HASONEVALUE: Validar si solo existe un valor en una columna. Ejemplo: Un Valor Exclusivo = HASONEVALUE('Ventas Videojuegos'[Año]) Funciones comunes y recomendadas

    7. Funciones comunes y recomendadas ADDCOLUMNS: Añadir columnas calculadas a una nueva tabla. Ejemplo: Completa Salarios = ADDCOLUMNS('Tabla Empleados', "Salario", SUM('Tabla Empleados'[Sueldo])) Funciones comunes y recomendadas

    8. Funciones comunes y recomendadas DISTINCT: Crear una tabla con valores distintos de una columna o tabla. Ejemplo: Lista de Departamentos = DISTINCT('Tabla Empleados'[Departamento]) Funciones comunes y recomendadas

    9. Funciones comunes y recomendadas GENERATESERIES: Devolver una tabla con una serie aritmética. Ejemplo: Tabla Series 0-100 = GENERATESERIES(0,100,10) Funciones comunes y recomendadas

    10. Funciones comunes y recomendadas SUMMARIZE: Devolver una tabla resumen de valores con generación de columnas. Ejemplo: Sueldos Totales por Jefe = SUMMARIZE('Tabla Empleados', 'Tabla Empleados'[Nombre Jefe], "Sueldo", SUM('Tabla Sueldo'[Sueldo])) Funciones comunes y recomendadas

    11. Funciones comunes y recomendadas TOPN: Devolver las N filas superiores de una tabla basado en agrupación y ordenación. Ejemplo: TOP Empleados mejores Evaluaciones = TOPN(10, 'Tabla Empleados', 'Tabla Empleados'[Evaluacion], DESC) Funciones comunes y recomendadas

    12. Funciones comunes y recomendadas SUMX: Devolver la suma de la expresión evaluada para cada fila de la tabla. Ejemplo: Suma de la Media de Sueldo por ID = SUMX('Tabla Sueldo', AVERAGE('Tabla Sueldo'[Sueldo])) Funciones comunes y recomendadas

    13. Funciones comunes y recomendadas AVERAGEX: Devolver la media de la expresión evaluada para cada fila de la tabla. Ejemplo: Media de la Suma de Sueldo por ID = AVERAGEX('Tabla Empleados',SUM('Tabla Sueldo'[Sueldo])) Funciones comunes y recomendadas

    14. Funciones comunes y recomendadas MAXX: Evaluar una expresión para cada fila y devolver el valor mayor. Ejemplo: Valor Máximo Sueldo Empleados = MAXX('Tabla Sueldo','Tabla Sueldo'[Sueldo]) Funciones comunes y recomendadas

    15. Funciones comunes y recomendadas MINX: Evaluar una expresión para cada fila y devolver el valor menor. Ejemplo: Valor Mínimo Sueldo Empleados = MINX('Tabla Sueldo','Tabla Sueldo'[Sueldo]) Funciones comunes y recomendadas

    20. Funciones para la creación de Datos CALENDAR: Devolver una tabla con fechas dentro de un rango especificado. Ejemplo: Tabla Calendario = CALENDAR("01-01-1990","31-12-2021") Funciones para la creación de Datos

    21. Funciones para la creación de Datos CALENDARAUTO: Devolver una tabla con fechas calculadas automáticamente según los datos del modelo. Ejemplo: Tabla Calendario = CALENDARAUTO() Funciones para la creación de Datos

    22. Funciones para la creación de Datos RAND(): Devolver un número decimal aleatorio entre 0 y 1. Ejemplo: Número Aleatorio = RAND() Funciones para la creación de Datos

    23. Funciones para la creación de Datos CONCATENATEX: Concatenar textos de una columna en una nueva columna calculada. Ejemplo: Nombre Completo = CONCATENATEX('Tabla Clientes', 'Tabla Clientes'[Nombre], ", ") Funciones para la creación de Datos

    24. Funciones para la creación de Datos UNION: Combinar dos tablas en una nueva tabla. Ejemplo: Tabla Combinada = UNION('Tabla A', 'Tabla B') Funciones para la creación de Datos

    25. Funciones para la creación de Datos ADDCOLUMNS: Añadir columnas calculadas a una nueva tabla. Ejemplo: Completa Sueldos = ADDCOLUMNS('Tabla Empleados', "Sueldo", SUM('Tabla Empleados'[Sueldo])) Funciones para la creación de Datos

    28. Funciones para la adición de Datos RELATED: Seguir relaciones y devolver el valor relacionado de otra tabla. Ejemplo: Categoría del Producto = RELATED('Tabla Categorías'[Categoría]) Funciones para la adición de Datos

    29. Funciones para la adición de Datos RELATEDTABLE: Seguir relaciones y devolver una tabla relacionada de otra tabla. Ejemplo: Productos Relacionados = RELATEDTABLE('Tabla Productos') Funciones para el filtrado de Datos

    30. Funciones para el filtrado de Datos Funciones para filtrar datos en tablas y mejorar la visualización. Uso de funciones como FILTER y ALL. Funciones para el filtrado de Datos

    31. Funciones para el filtrado de Datos FILTER: Filtrar filas de una tabla basándose en una condición. Ejemplo: Ventas 2022 = FILTER('Ventas', 'Ventas'[Año] = 2022) Funciones para el filtrado de Datos

    32. Funciones para el filtrado de Datos ALL: Eliminar todos los filtros de una columna o tabla. Ejemplo: Ventas Totales = CALCULATE(SUM('Ventas'[Monto]), ALL('Ventas'[Producto])) Funciones para el filtrado de Datos

    33. Funciones para el filtrado de Datos VALUES: Devolver una tabla con los valores distintos de una columna. Ejemplo: Valores Únicos de Producto = VALUES('Ventas'[Producto]) Funciones para el filtrado de Datos

    34. Funciones para el filtrado de Datos CALCULATETABLE: Evaluar expresiones en un contexto modificado por filtros. Ejemplo: Ventas Altas 2022 = CALCULATETABLE('Ventas', 'Ventas'[Monto] > 1000) Funciones para el filtrado de Datos
