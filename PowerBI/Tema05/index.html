    Tema 5: Preparación de los Datos

    Resumen

    • Capítulo 1 – Personalización de Tablas • Capítulo 2 – Transformación de Datos • Capítulo 3 – Agrupación de Columnas • Capítulo 4 – Anidación de Columnas • Capítulo 5 – Eliminación de Errores • Capítulo 6 – Otras operaciones avanzadas sobre columnas • Capítulo 7 – Creación de Tablas Calculadas • Capítulo 8 – Combinación de Consultas

    La preparación de datos es esencial para garantizar que la información cargada esté limpia, formateada y adaptada a los objetivos del proyecto. En este tema, se aborda la Personalización de Tablas utilizando Power Query, una herramienta integrada en Power BI. La fase de construcción del dashboard se denomina Data Preparation (DP), enfocándose en dar formato, limpiar y preparar los datos para cumplir con los requisitos del proyecto y controlar errores.

    Capítulos

    1. Personalización de Tablas (Capítulo 21)

    • Una vez cargados los datos, se utiliza Power Query para adaptarlos.
    • Power Query, con su entorno amigable, es potente al usar el lenguaje "M" para preparar datos paso a paso.

    2. Objetos Visuales en Power Query (Capítulo 61)

    • Se explican los objetos visuales en Query Editor para familiarizar a los alumnos con Power Query.
    • Se presentan grupos como 'Inicio', 'Transformar' y 'Agregar Columna', esenciales para la preparación del dato.

    3. Funciones Básicas en Power Query (Capítulo 131)

    • Funciones del grupo 'Inicio': Administrar Columnas, Reducir Filas, Ordenación, Dividir Columnas, Agrupar Filas, Especificar Tipo de Datos, Usar la primera fila como encabezado, Reemplazar Valores.

    4. Transformación de Datos (Capítulos 222-402)

    • Se detallan funciones clave para transformar datos, como Eliminar espacios en blanco, Trasponer tabla, Invertir filas, Rellenar, Cambiar Nombre, Convertir en Lista, Estadísticas, Operaciones Estándar, Científicas, Trigonométricas, Redondeo, Información, Fecha, hora o duración, y Scripts de R y Python.

    5. Agrupación de Columnas (Capítulos 423-473)

    • Se introduce la función de Agrupar por en Power Query para combinar valores de columnas.
    • Se explica cómo realizar agrupaciones y se proporciona un ejemplo de DAX para agrupar ventas totales en Japón.

    6. Anidación de Columnas (Capítulos 484-523)

    • Se presenta la anidación de columnas, que permite incluir tablas dentro de tablas.
    • La función Combinar Tablas se utiliza para conectar fuentes de datos y consolidar en una única consulta.

    7. Eliminación de Errores (Capítulos 534-635)

    • Se aborda la limpieza de datos y la importancia de minimizar riesgos basados en información imprecisa.
    • Se describen problemas relacionados con datos de una única fuente y datos de diversas fuentes.
    • Fases del Data Cleansing: Análisis de datos, Definición del flujo de transformación y reglas de mapeo, Verificación, Transformación, Reflujo de datos limpios.

    Operaciones Avanzadas sobre Columnas en Power Query

    Funciones avanzadas en Power Query

    • Dentro de Power Query, existen funciones avanzadas para operar en columnas de tablas.
    • Las funciones tienen componentes visuales en forma de botón, pero también se pueden extender con el lenguaje "M".

    Funciones destacadas en el grupo "Agregar Columna"

    Columnas a partir de ejemplos

    • Permite generar una columna a partir de selecciones de otras columnas, proponiendo máscaras para crear nuevos valores.

    Columna Personalizada

    • Abre un editor de fórmulas para generar nuevas columnas calculadas a partir de campos de la tabla de origen.

    Invocar función personalizada

    • Requiere la creación previa de una función personalizada para su uso en la tabla objetivo, facilitando la reutilización de funciones.

    Creación de Tablas Calculadas

    • Permite agregar nuevas tablas basadas en datos ya cargados mediante fórmulas DAX.
    • Se utilizan para cálculos, obtener datos intermedios y establecer relaciones con otras tablas.
    • Las tablas calculadas se recalculan al actualizarse las tablas de origen, a menos que utilicen DirectQuery.

    Combinación de Consultas en Power Query

    Combinar Consultas

    • Permite unir diferentes tablas, ya sea ampliando el número de columnas o aumentando el número de filas.
    • Tipos de combinación: Externa izquierda, Externa derecha, Externa completa, Interna, Anti-izquierda, Anti-derecha.

    Anexar consultas

    • Inserción de filas en lugar de columnas, con dos tipos: Anexar en la misma tabla "Origen" o en una nueva tabla.
    • Concatenación de filas de dos o más tablas sin relacionar datos.

    Combinar archivos

    • Selecciona una carpeta con archivos para combinar, filtra los archivos y realiza la combinación según campos comunes.

    Tablas Dinámicas en Power BI

    • Reportes flexibles que permiten cambiar fácilmente las columnas y filas visualizadas.
    • Ideal para cambiar la segmentación por valores en filas sin necesidad de fórmulas complejas.

    Dinamización de Tablas

    • Permite convertir filas en columnas y seleccionar la operación a realizar.
    • No se permite el uso de tablas anidadas.

    Anulación de Dinamización

    • Permite revertir la dinamización de columnas, generando pares de columnas del tipo Atributo-Valor.

    Nota para los alumnos: Estas funciones y técnicas son fundamentales para la preparación de datos en Power BI y el modelado de información para su representación en los cuadros de mando. Se recomienda practicar y explorar estas herramientas para un mejor entendimiento y aplicación práctica.
