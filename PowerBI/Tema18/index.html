    1. Formas de publicar Informes

    En este punto, se verá cómo se publica un cuadro de mando en Power BI Service. Se detallarán diferentes formas de publicar y compartir informes una vez estén en Power BI Service.

    Una vez aplicadas las buenas prácticas de desarrollo, revisado el rendimiento y definidos los roles de segmentación de datos, se procederá a la publicación en Power BI Service.

    Pasos para la publicación

    1. Publicar:
    2. Seleccionar el recurso de publicar en Power BI Desktop para iniciar el proceso de publicación.

    Nota: La primera vez que se ejecute esta función, se solicitarán las credenciales de acceso a Power BI Service.

    1. Seleccionar Área de Trabajo:
    2. Tras seleccionar la función de "Publicar" y estar logueados, se elige el área de trabajo donde se desea publicar el tablero objetivo.

    Nota: Se pueden crear nuevas áreas en Power BI Service si es necesario.

    1. Finalizar el Proceso:
    2. Esperar a que finalice el proceso de publicación.
    3. Una vez completado, se puede validar la subida en Power BI Service.

    Formas de compartir

    Power BI permite compartir informes de diversas maneras, dependiendo de la licencia del usuario.

    • Usuarios sin licencia:
    • Crear un archivo .pbix

    • Exportar un informe en PDF

    • Usuarios con licencia:

    • Compartir vínculo (Personas de la organización y Personas específicas)
    • Compartir a través de un sitio web o portal
    • Incrustar reportes en SharePoint Online
    • Compartir a través de áreas de trabajo
    • Compartir a través de aplicaciones

    2. Creación de Paneles

    Se diferencia entre informes y paneles. Un informe es el resultado de datos visualizados, mientras que un panel es una página única que cuenta una historia a través de visualizaciones.

    Diferencias entre Informes y Paneles

    Informes: - Páginas múltiples. - Se pueden crear en Desktop y Service. - Un conjunto de datos por informe. - Admite varias opciones de filtro. - Se pueden establecer como informes destacados.

    Paneles: - Página única. - Se crea solo en Service. - Puede usar varios informes y conjuntos de datos. - Admite marcadores como filtro. - Se puede establecer como panel destacado.

    Creación de Paneles

    1. Acceder al área de trabajo:

    2. Seleccionar el área de trabajo.

    3. Seleccionar nuevo panel:

    4. Pinchar en el botón "Nuevo > Panel" bajo el título del área seleccionada.

    5. Definir el nombre del panel:

    6. Especificar el nombre del nuevo panel.

    7. Anclar objetos en el panel:

    8. Desde los informes, anclar objetos visuales al nuevo panel.

    3. Gestión y Limitaciones al Publicar en Áreas de Trabajo

    Las áreas de trabajo tienen funciones limitadas. Se pueden agregar o quitar miembros y editar el nombre y anotaciones del área.

    Limitaciones

    • Máximo 1000 conjuntos de datos o informes por conjunto de datos.
    • Almacenamiento limitado (10 GB en áreas de trabajo compartidas).

    4. Gestión de Permisos de Acceso a Datos

    Configurar permisos en áreas de trabajo para definir quién puede ver, modificar o crear informes y conjuntos de datos.

    2. Seleccionar el área a la que se le quieren asignar usuarios y nivel de permisos.

    3. Seleccionar permisos:

    4. Hacer clic en "Acceso" para definir usuarios y asignar niveles de acceso.

    5. Introducir emails y roles:

    6. Completar los correos de los usuarios y asignar roles.
