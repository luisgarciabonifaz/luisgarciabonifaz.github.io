    Tema 19: Instalación de un Data Gateway

    1. Entendiendo la necesidad de un Data Gateway

    Power BI es una herramienta para convertir datos en conocimiento, facilitando la toma de decisiones empresariales rápidas e informadas. En este capítulo, se aborda la necesidad de un Data Gateway para acceder a datos de manera dinámica desde ubicaciones fuera de la red.

    2. Instalación

    Se detallan los pasos para la instalación del Data Gateway, asegurando su correcto funcionamiento en un equipo estable y con buena conectividad.

    2.1 Descargar software de puerta de enlace

    • Enlace de descarga: Data Gateway de Microsoft

    2.2 Definir ruta y aceptar términos de uso y privacidad

    • Seleccionar la ruta de instalación y aceptar los términos y condiciones legales.

    2.3 Introducir la cuenta de correo Office365 para iniciar sesión

    • Ingresar la cuenta de correo asociada a Office365 para registrar la configuración del Data Gateway.

    2.4 Registrar nueva puerta de enlace o migrar/restaurar

    • Seleccionar la opción adecuada según se desee registrar una nueva puerta de enlace o migrar/restaurar una existente.

    2.5 Definir el nombre del Data Gateway

    • Establecer un nombre único y clave de recuperación para la puerta de enlace.

    2.6 Revisar la configuración de la instalación

    • Verificar la configuración antes de completar la instalación.

    3. Configuración

    Tras la instalación, se explica cómo configurar el Data Gateway en el equipo.

    3.1 Abrir la aplicación de la puerta de enlace

    • Buscar "On-premises" en el buscador de Windows y abrir la aplicación como administrador.

    3.2 Configuración de Servicio, Red, Conectores, etc.

    • Se describen las opciones de configuración disponibles, incluyendo la configuración del servicio, la configuración de red y la gestión de conectores.

    4. Selección del Data Gateway y desde Power BI Cloud

    Se detalla cómo seleccionar un Data Gateway desde Power BI Service y administrarlos.

    4.1 El Data Gateway como aplicación de servicio

    • Explicación de cómo el Data Gateway funciona como servicio, actualizando la información del DataSet a la versión en Power BI Service.

    4.2 Configuración de la puerta de enlace desde Power BI Cloud

    • Se explica cómo configurar la puerta de enlace desde Power BI Service, asignándola a un conjunto de datos y gestionando las credenciales.

    4.3 Administración de las puertas de enlace desde Power BI Cloud

    • Se detallan los pasos para administrar las puertas de enlace desde Power BI Service, incluyendo la creación de una cuenta On-premises Data Gateway de Microsoft.
